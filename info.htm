Auroracoin QR-CODE calculator

 

This is a simple PHP based QRcode calculator designed for AURORACOIN. The calculator was built to eaasily convert amount in ISK (the Icelandic Króna)  to Auroracoin value and display a QR code that can be scanned by any mobile wallet.

The API can be used to display QRcode in Auroracoin straight, or calculated from ISK. The price is calculated from the current price on ISX, the Icelandic Auroracoin currency  exchange market.

 It is based on two files,  index.php and qrcodegen.php.

Main use:

The index.php in the main directory can be called with no parameters, but you need to paste the address into the input field to use it. When only called with an address, it will generate a neautral QR code (no value)

It will take the following GET statements:

address=(Auroracoin address)

isk=(amount in icelandic Króna)

addressfield=(yes, no) Default is yes. Set to anything else to hide the address input field. Use only when also using .address. Get statement)

 

Here is an example:
http://qrcode.auroracoin.tools/?address=AcM8V615AM3dj1Xd93W43RrBMPFznggTgJ&addressfield=no&isk=500  (this is my Auroracoin Address, please concider a donation if you like my creation).

 

API:

The qrcodegen.php is used to generate pure PNG image. It can then be inserted anywhere you like. the only mandatory Get field is .address., and if called with that only, it will generate a neutral QRcode (no value).

It will take the following GET statements:

address=(Auroracoin address) This is mandatory. All other fields are optional.

isk=(amount in icelandic Króna) This will be calculated to AUR amount using ISX exchange.      

aur=(amount in Auroracoin) This will be displayed straight up with no calculation as AUR. If both .isk. and .aur. statements are used, .aur. will be displayed. If both are missing, a neutral QRcode is generated.

qrsize=(width in pixels) defaults to 200. Anything below 50 is hard to scan and everything above 2000 is plain stupid :)

showlabel=(yes, no) Defaults to yes. If set to anything else, the label code below the QRcode will be hidden.

 

Here is an example:
http://qrcode.auroracoin.tools/qrcodegen.php?aur=25&walletaddress=AcM8V615AM3dj1Xd93W43RrBMPFznggTgJ&qrsize=400&showlabel=yes (this is my Auroracoin Address, please concider a donation if you like my creation).

 

 

You can use plain html code to call my image like this:

<img src="qrcode.auroracoin.tools/qrcodegen.php?aur=25&walletaddress=AcM8V615AM3dj1Xd93W43RrBMPFznggTgJ">
